Teste de Webservices SOAP
SOAP webservice tester

Utiliza os módulo express e node-soap

Essa é uma interface para teste de conexão a webservices SOAP wdsl por aplicações NodeJS.

Para utilizar, informe na tela a seguir, a url da webservice, incluindo a parte do wsdl, o nome do método a ser testado e os parâmetros de entrada, em formato json, com os mesmos nomes dos parâmetros do método da webservice a ser testada.

Uses node-soap and express packages

This is an template application to test SOAP webservice's functions from a NodeJs Server.

To use it, insert into next page the following information:
webservice url - including that wsdl part,
function name - CaseSensitive func tion's name,
input params - JSON formatted input params names and values
